About This Project

This PHP project demonstrates how to send emails using PHPMailer with SMTP authentication. It uses a secure email delivery system and environment-based credentials.

Features

  • SMTP-based Email Sending
  • Environment Variables Support (.env)
  • Simple, Secure, and Reliable
  • Ready for Deployment
